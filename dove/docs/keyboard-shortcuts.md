# Keyboard Shortcuts

Active on the main page when not typing in a form field.

| Key | Action |
|-----|--------|
| `1` – `9` | Select scene by number |
| `Enter` | Cut selected scene to program |
| `i` | Add Input |
| `o` | Add Output |
| `s` | Add Scene |
| `e` | Add Encoder |
| `?` | Open this help page |
